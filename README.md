# Marketing Campaign Performance Analysis

## Overview
This project analyzes customer demographic and purchase behavior data to predict marketing campaign conversion using machine learning techniques.

## Dataset
Marketing Campaign Dataset (Kaggle)

## Objective
To predict whether a customer will respond to a marketing campaign and identify key factors influencing campaign success.

## Project Structure
```   
marketing-campaign-performance-analysis/
├── data/
│   └── marketing_campaign.csv
├── notebooks/
│   └── marketing_campaign_analysis.ipynb
├── images/
│   ├── confusion_matrix.png
│   └── model_results.png
├── README.md
└── requirements.txt
```   

## Requirements
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

Install dependencies using:

pip install -r requirements.txt

## Methodology
- Data loading and preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling
- Logistic Regression model training
- Model evaluation using accuracy and confusion matrix

## Results
The model helps identify customers who are more likely to respond to marketing campaigns, enabling better targeting and reduced marketing costs.

## How to Run
1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Open the notebook in the `notebooks/` folder
4. Run all cells sequentially
